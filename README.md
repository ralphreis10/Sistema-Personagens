# Projeto Integrador - Sistema de Personagens

**Status:** Em desenvolvimento

**Tecnologias utilizadas:**
- Java SE
- Swing (Interface gráfica)
- JDBC (conexão com banco MySQL)
- Git/GitHub (versionamento)

**Time de desenvolvedores:**
- Ralph

**Objetivo do software:**
Gerenciar personagens e suas informações, permitindo cadastro, edição, listagem e filtro por características.

**Funcionalidades do sistema:**
- Cadastro de personagens com atributos: Nome, Vida, Raça, Idade
- Edição de personagens existentes
- Listagem de personagens em tabela
- Filtro por nome ou raça
- Armazenamento em banco de dados MySQL
